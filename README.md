### Idee:
1. Sensori (giroscopio), rappresentazione vituale del braccio tramite un software.
2. Telacamera dietro la pinza per foto (A.I. per riconoscere oggetti).
3. A.I. per giocare a tris.
</br></br>



### Numero motori:
1. Aprire/Chiudere pinza. (limite: sensore pressione)
2. Ruotare la pinza. (no limiti)
3. Polso (limite: angolo)
4. Gomito (limite: angolo)
5. Spalla (limite: angolo)
6. Ruotare tutto il braccio (no limiti)
